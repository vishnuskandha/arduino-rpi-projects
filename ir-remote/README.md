# IR Remote Receiver

Read IR remote signals with Arduino and act on button presses.

## Components
- Arduino Uno/Nano
- IR receiver (VS1838B)
- Remote control

## Arduino
rduino/IR_remote.ino

## Run
- Wire receiver OUT to a digital pin
- Upload and open Serial Monitor

## Tested hardware
- Arduino Uno/Nano
- VS1838B IR receiver
- IR remote control

## Circuit mapping
- IR OUT -> Arduino digital pin (match sketch)
- VCC->5V, GND->GND
