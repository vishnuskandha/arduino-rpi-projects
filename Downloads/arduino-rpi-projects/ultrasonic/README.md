# Ultrasonic Distance

Measure distance using HC-SR04 ultrasonic sensor.

## Components
- Arduino Uno/Nano
- HC-SR04 sensor

## Arduino
rduino/ultrasonic.ino

## Run
- Wire TRIG/ECHO per sketch
- Upload and read Serial Monitor for distance

## Tested hardware
- Arduino Uno/Nano
- HC-SR04 ultrasonic sensor

## Circuit mapping
- Trig -> D10, Echo -> D9
- VCC->5V, GND->GND
